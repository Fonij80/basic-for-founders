# Django for APIs

## Definitions

- API-first approach: separating back-end from front-end

- **Virtual Environment**: allow you to create and manage separate environments for each Python project on the same computer

```bash
python3 -m venv .venv  # creating a VE using venv (which is part of the Python3 standard lib)
```

```bash
source .venv/bin/activate  # activate the VE
```

```bash
deactivate  # deactivate the VE
```

```bash
python -m pip install --upgrade pip  # upgrade pip to latest version
```

### Django Installation

```bash
python -m pip install django  # install the latest version of Django
```

```bash
python -m pip install django-rest-framework  # install the latest version of DRF
```

### Setup Python Formatter for VScode

```bash
python -m pip install black
```

- search for "python formatting provider" in VScode Settings and choose black
